# 瑞昌藥局門市人時生產力預警與人力成本加權分析工具

> **Tier 1 項目** - 陳依萍（總部-人資部）提案

## 🎯 系統概述

本系統解決瑞昌藥局人資部門的核心痛點：將「月底事後檢討」轉變為「月中即時預警」，透過智能加權計算實現精準的門市生產力監控。

## 📋 核心功能

### ✅ 已實現功能

1. **兼容性輸入介面**
   - 店長友好的工時營收輸入表單
   - 支援8家門市數據管理

2. **智能加權計算引擎**
   - 年資權重：≥3年員工權重1.2倍
   - 職級權重：店長(1.3)、副店長(1.15)、藥師(1.2)、正職(1.0)、兼職(0.85)
   - 自動計算標竿人時生產力

3. **即時預警看板**
   - 紅黃綠燈視覺化警示系統
   - 生產力達成率監控
   - 人力成本比率分析

4. **完整API架構**
   - RESTful設計，預留未來統一排班系統串接
   - 完整的CRUD操作支援

## 🚀 部署說明

### 環境需求
- Node.js >= 16.0.0
- SQLite 3

### 本地開發
```bash
# 安裝依賴
npm install

# 初始化資料庫
npm run init-db

# 啟動開發伺服器
npm run dev
```

### 生產部署 (Zeabur)
```bash
# 設置環境變數
NODE_ENV=production
BASE_PATH=/productivity
PORT=3000

# 啟動
npm start
```
